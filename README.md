# My Personal Project

## A meal prep helper
 
A question that I ask both myself and my parents all the time is *"What's for dinner?"*
Usually, this question is answered with a sad *"I don't know"*, and this project is meant to fix that.
This project would allow the user to add recipes and food in the house to a collection of data, and then help 
the user pick a meal to make. 

Choices could be made from:
- What ingredients are in the house?
- What kind of food does the user want to eat?
- How much time does the user have? 

This project is meant for my own use, as well as **anyone** who has trouble deciding what to eat. It is meant to be
implemented into **daily life**, making a common decision in life much easier. 

## User Stories
- I want to be able to add a dish to my dish collection and be able to specify what is needed to make it, 
how to make it, and what kind of dish it is
- I want to be able to view a list of dishes that fit a specific category (cuisine/speed/vibes)
- I want to be able to be recommended a dish based on a certain categories
- I want to be able to be recommended ingredients to buy based on what I want to eat
- I want to be able to save my recipes
- I want to be able to load my recipes

## Instructions for End User

- **Run RecipeAppGui to start**
- To view recipes, click view
- to add a recipe, click add, then type info into fields and then add recipe
- To view recipe details, click recipe in recipe viewing mode then click view recipe
- To see visuals, look at the bottom half of the application at the bottom half of the screen
- To load recipes, click save/load then click load
- To save recipes, click save/load then click save
- To get a suggestion, click get suggestions then type descriptors in box
- note: make sure to save/load data before using/closing app
